# TinyApp 

A tiny URL shortener application writen with nodejs/express

## How to run

/tiny-app directory:

npm start 


## Dependencies
- express js
- [EJS template engine](https://ejs.co/)
- cookie-session
- bcryptjs
- (dev) mocha/chai
- (dev) nodemon https://github.com/remy/nodemon
